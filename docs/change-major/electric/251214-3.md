## 笔试
1. 高数 20分  
内容：教材前两章，考到积分之前  
整体偏基础，难度不超过高数期中考  
备考：其实提前翻阅一下课本，然后课上老师节奏走课下认真完成作业就差不多了，如果不放心可以刷吉米多维奇
2. 线代 10分  
内容：教材前三章  
只考了四题，有关于矩阵秩的概念辨析的，我不太确定有没有做对  
备考：有学线代的就跟着老师节奏走就好，没学线代的就看看福大的线代课本，刷吉米多维奇
3. 高中物理 50分  
内容：主要靠电磁学，都是选择题  
基本都是靠基础公式概念  
涉及库伦定律，焦耳热，静电屏蔽，交流电的原理。。。（25年没怎么考电磁感应，什么动量定律，能量守恒那些）  
备考：我是刷完了电磁学必刷题，紫色的那本，还有几套高考题，但说实话，建议系统的复习电磁学，刷那套必刷题对于这次转专业来说没啥用，很多知识点没覆盖，导致我考物理的时候很懵，笔试成绩比较低，因为这次考试没有复杂的题型，主要是一些基础的、细节的知识点。
4. 英语翻译 20分  
内容：考了新能源，可持续发展，人工智能等等的，好像都和十五五规划内容有关，对于高中英语基础比较好的应该没什么问题
备考：建议适当关注实事的英文表达，考前适当做点题保持语感

## 面试
25年是桌面上有五个信封五套题，你自己抽一套，每套题里有五道题，当你打开信封时计时开始，限时五分钟，剩下一分钟时会提醒
我当时抽到的题是：

1. 自我介绍
2. 转入电气后会面临很重的学业压力，你将怎么规划时间和管理压力呢？
3. 为什么电流经过氖管时氖管会发亮
4. 如何理解“无体育，不电气”
5. 你认为家里客厅和卧室的灯是并联还是串联的，为什么？
