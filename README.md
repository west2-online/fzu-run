# 福州大学飞跃手册

福州大学飞跃手册，福州大学学生自己的转专业、保研/考研经验交流站点

## Goal

对于985/211、双一流高校学生而言,专业的选择愈发重要。

不论是转专业亦或是保研、考研和出国，学生们面对的都是一场充满未知的信息战。

我们深知信息差带来的困扰，因此，我们致力于提供一个文档站点，通过成功上岸的学长学姐的经验以及历年的资料，弥补信息差，为每一位福大学子提供尽可能明晰的方向指导。
